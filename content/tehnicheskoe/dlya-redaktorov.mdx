# Для редакторов

Эта страница создана для того, чтобы помочь редакторам понять основы форматирования статей в нашей документации. Мы используем современный формат MDX, который позволяет смешивать обычный текст и интерактивные компоненты.

## Базовое форматирование

Вы можете использовать **жирный текст**, *курсив*, ~~зачеркнутый текст~~ или `встроенный код`.

### Списки

Маркированный список:
- Первый пункт
- Второй пункт
  - Подпункт А
  - Подпункт Б

Нумерованный список:
1. Шаг первый
2. Шаг второй

## Изображения

Все картинки автоматически оборачиваются в современный просмотрщик (Lightbox) с возможностью приближения и полноэкранного режима.

Просто используйте стандартный синтаксис:

```markdown
![Описание картинки](/media/tutorial/image.jpg)
```

![Тестовая картинка](/media/tutorial/image.jpg)

## Видеоплеер

Для вставки видео мы используем наш кастомный видеоплеер. Он автоматически поддерживает извлечение обложки и имеет локализацию.

```tsx
<VideoPlayer src="/media/tutorial/video.mp4" title="Тестовое видео" />
```

## Спойлеры и аккордеоны

Если нужно скрыть какую-то длинную информацию, используйте HTML-тег `details`:

```html
<details>
  <summary>Нажмите, чтобы раскрыть секретную информацию</summary>
  
  Здесь находится скрытый контент. Вы можете использовать **форматирование** внутри него!
</details>
```

## Блоки кода

Для демонстрации кода мы используем плагин Code Hike. Просто добавьте название файла после языка:

````markdown
```javascript ! app.js
console.log("Привет, AfricaRP!");
```
````

```javascript ! app.js
console.log("Привет, AfricaRP!");
```

Вы также можете выделять конкретные строки (удалите пробел между `//` и `!mark` при использовании):

````markdown
```javascript ! server.js
const express = require('express');
const app = express(); // ! mark

app.listen(3000);
```
````

```javascript ! server.js
const express = require('express');
const app = express(); // !mark

app.listen(3000);
```
