# Для редакторов

Здесь собраны все элементы форматирования, доступные при написании статей.

## Текст

Для выделения текста используйте следующие конструкции:

```markdown
**Жирный текст**
*Курсив*
~~Зачеркнутый текст~~
`встроенный код`
[Ссылка на сайт](https://africa-rp.su)
```

**Жирный текст**
*Курсив*
~~Зачеркнутый текст~~
`встроенный код`

## Таблицы

Для создания таблиц используйте вертикальные черты `|` и дефисы `-`:

```markdown
| Колонка 1 | Колонка 2 | Колонка 3 |
| :--- | :---: | ---: |
| Левый край | По центру | Правый край |
| Текст | Текст | Текст |
```

| Колонка 1 | Колонка 2 | Колонка 3 |

| Текст | Текст | Текст |

## Блоки кода

Вставка многострочного кода. Добавьте название файла после языка:

````markdown
```javascript ! app.js
console.log("Привет, AfricaRP!");
```
````

```javascript ! app.js
console.log("Привет, AfricaRP!");
```

Подсветка конкретных строк (удалите пробел между `//` и `!mark`):

````markdown
```javascript ! server.js
const express = require('express');
const app = express(); // ! mark

app.listen(3000);
```
````

```javascript ! server.js
const express = require('express');
const app = express(); // !mark

app.listen(3000);
```
