# Для редакторов

Здесь собраны все элементы форматирования, доступные при написании статей. Мы подготовили наглядные примеры по принципу «Как писать» -> «Как выглядит».

## Текст

**Как писать:**
```markdown
**Жирный текст**
*Курсив*
~~Зачеркнутый текст~~
`встроенный код`
[Ссылка на сайт](https://africa-rp.su)
```

**Как выглядит:**

**Жирный текст**
*Курсив*
~~Зачеркнутый текст~~
`встроенный код`
[Ссылка на сайт](https://africa-rp.su)

## Списки

**Как писать:**
```markdown
Маркированный список:
- Первый пункт
- Второй пункт
  - Подпункт А
  - Подпункт Б

Нумерованный список:
1. Шаг первый
2. Шаг второй
```

**Как выглядит:**

Маркированный список:
- Первый пункт
- Второй пункт
  - Подпункт А
  - Подпункт Б

Нумерованный список:

2. Шаг второй

## Таблицы

Для создания таблиц используйте вертикальные черты `|` и дефисы `-`:

**Как писать:**
```markdown
| Колонка 1 | Колонка 2 | Колонка 3 |
| :--- | :---: | ---: |
| Левый край | По центру | Правый край |
| Текст | Текст | Текст |
```

**Как выглядит:**

| Колонка 1 | Колонка 2 | Колонка 3 |
| :--- | :---: | ---: |
| Левый край | По центру | Правый край |
| Текст | Текст | Текст |

## Многострочный код

Для создания блоков кода просто оберните текст в тройные обратные кавычки и укажите язык.
Чтобы добавить красивый заголовок с названием файла, напишите `! имя_файла.js` сразу после языка.

**Как писать:**
````markdown
```javascript ! app.js
console.log("Привет, AfricaRP!");
```
````

**Как выглядит:**

```javascript ! app.js
console.log("Привет, AfricaRP!");
```

## Подсветка строк кода

Вы можете добавлять красивые всплывающие пояснения к строкам кода. Для этого добавьте специальный комментарий (удалите пробел после восклицательного знака): `// ! callout[/слово/] ваш текст`.

**Как писать:**
````markdown
```javascript ! server.js
const express = require('express');
const app = express(); // ! callout[/express/] Инициализация приложения

app.listen(3000);
```
````

**Как выглядит:**

```javascript ! server.js
const express = require('express');
const app = express(); // !callout[/express/] Инициализация приложения

app.listen(3000);
```
