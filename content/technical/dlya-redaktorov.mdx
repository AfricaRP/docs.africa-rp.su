# Для редакторов

Здесь собраны все элементы форматирования, доступные при написании статей. Мы подготовили наглядные примеры по принципу «Как писать» -> «Как выглядит».

## Текст и клавиши

Вы можете использовать стандартное оформление текста, а также специальный тег `<Kbd>` для отображения клавиш клавиатуры.

**Как писать:**
```mdx
**Жирный текст**
*Курсив*
~~Зачеркнутый текст~~
`встроенный код`
Нажмите <Kbd>Alt</Kbd> + <Kbd>F4</Kbd>
[Ссылка на сайт](https://africa-rp.su)
```

**Как выглядит:**

**Жирный текст**
*Курсив*
~~Зачеркнутый текст~~
`встроенный код`
Нажмите <Kbd>Alt</Kbd> + <Kbd>F4</Kbd>
[Ссылка на сайт](https://africa-rp.su)

## Списки

  - Подпункт А
  - Подпункт Б

Нумерованный список:

**Как выглядит:**

- Первый пункт
- Второй пункт
  - Подпункт А
  - Подпункт Б

Нумерованный список:

2. Шаг второй

## Таблицы

Для создания таблиц используйте вертикальные черты `|` и дефисы `-`:

**Как писать:**
```markdown
| Колонка 1 | Колонка 2 | Колонка 3 |
| :--- | :---: | ---: |
| Левый край | По центру | Правый край |
| Текст | Текст | Текст |
```

**Как выглядит:**

| Колонка 1 | Колонка 2 | Колонка 3 |
| :--- | :---: | ---: |
| Левый край | По центру | Правый край |
| Текст | Текст | Текст |

## Изображения и Сравнения

Вы можете вставлять обычные картинки, а также интерактивный слайдер "Было/Стало" (тег `<ImageCompare>`).

**Как писать:**
```mdx
![Описание картинки](/media/tutorial/image.jpg)

<ImageCompare 
  before="/media/tutorial/image.jpg" 
  after="/media/tutorial/image.jpg" 
  beforeLabel="Без модов"
  afterLabel="С редуксом"
/>
```

**Как выглядит:**

![Описание картинки](/media/tutorial/image.jpg)

<ImageCompare 
  before="/media/tutorial/image.jpg" 
  after="/media/tutorial/image.jpg" 
  beforeLabel="Без модов"
  afterLabel="С редуксом"
/>

## Вкладки (Tabs)

Удобно для разделения контента, например, для разных ОС.

**Как писать:**
```mdx
<Tabs>
  <Tab title="Windows">
    Инструкция для Windows.
  </Tab>
  <Tab title="macOS">
    Инструкция для macOS.
  </Tab>
</Tabs>
```

**Как выглядит:**

<Tabs>
  <Tab title="Windows">
    Инструкция для Windows.
  </Tab>
  <Tab title="macOS">
    Инструкция для macOS.
  </Tab>
</Tabs>

## Дерево файлов

Визуализируйте структуру папок. Указывайте расширение файла (`.txt`, `.js`), чтобы отобразилась иконка документа, или пишите без расширения, чтобы отобразилась иконка папки.

**Как писать:**
```mdx
<FileTree>
- src
  - components
    - Button.tsx
    - Card.tsx
  - index.ts
- package.json
</FileTree>
```

**Как выглядит:**

<FileTree>
- src
  - components
    - Button.tsx
    - Card.tsx
  - index.ts
- package.json
</FileTree>

## Всплывающие подсказки (Глоссарий)

Используйте `<Tooltip>` для объяснения терминов при наведении.

**Как писать:**
```mdx
На сервере запрещен <Tooltip text="Random Deathmatch — убийство без причины">RDM</Tooltip>.
```

**Как выглядит:**

На сервере запрещен <Tooltip text="Random Deathmatch — убийство без причины">RDM</Tooltip>.

## Блоки внимания (Callouts)

Используйте `<Callout>` для выделения важной информации. Доступные типы (type): `note` (синий, по умолчанию), `tip` (зеленый), `warning` (желтый), `danger` (красный).

**Как писать:**
```mdx
<Callout type="tip" title="Полезный совет">
  Всегда пристегивайте ремень безопасности в машине!
</Callout>

<Callout type="danger" title="Важно">
  За использование читов выдается перманентный бан.
</Callout>
```

**Как выглядит:**

<Callout type="tip" title="Полезный совет">
  Всегда пристегивайте ремень безопасности в машине!
</Callout>

<Callout type="danger" title="Важно">
  За использование читов выдается перманентный бан.
</Callout>

## Карточки навигации (Card Grid)

Используйте сетку карточек для красивого оформления индексных страниц разделов.

**Как писать:**
```mdx
<CardGrid>
  <Card title="Правила сервера" href="/rules" icon="Shield">
    Ознакомьтесь с основными правилами проекта.
  </Card>
  <Card title="Начать игру" href="/start" icon="Play">
    Инструкция по установке лаунчера.
  </Card>
</CardGrid>
```

**Как выглядит:**

<CardGrid>
  <Card title="Правила сервера" href="/rules" icon="Shield">
    Ознакомьтесь с основными правилами проекта.
  </Card>
  <Card title="Начать игру" href="/start" icon="Play">
    Инструкция по установке лаунчера.
  </Card>
</CardGrid>

## Многострочный код

Для создания блоков кода оберните текст в тройные обратные кавычки и укажите язык.
Чтобы добавить красивый заголовок с названием файла, напишите `имя_файла.js` сразу после языка.

**Как писать:**
````markdown
```javascript app.js
console.log("Привет, AfricaRP!");
```
````

**Как выглядит:**

```javascript app.js
console.log("Привет, AfricaRP!");
```

## Подсветка и фокус строк кода

Вы можете выделять строки (`// !mark`) или фокусировать внимание только на нужных строках, затемняя остальные (`// !focus`).

**Как писать:**
````markdown
```javascript server.js
const express = require('express'); // !focus
const app = express(); // !mark

app.listen(3000);
```
````

**Как выглядит:**

```javascript server.js
const express = require('express'); // !focus
const app = express(); // !mark

app.listen(3000);
```
