# Медиа файлы

В документации AfricaRP вы можете легко вставлять видеоролики и картинки. Все медиа-элементы поддерживают удобный и современный просмотр.

## Демонстрация картинок

Все картинки в документации автоматически становятся "кликабельными". Если вы нажмете на картинку, она откроется в красивом современном просмотрщике (Lightbox) с возможностью приближения (зума) и полноэкранного режима!

Просто кликните на эту картинку:

![Красивый пейзаж AfricaRP](/media/image.jpg)

Чтобы вставить картинку, используйте стандартный синтаксис Markdown:
```md
![Описание картинки](/media/ваша-картинка.jpg)
```

## Как использовать видео

Чтобы добавить такое же видео на любую другую страницу, достаточно использовать один простой тег:

```jsx
<VideoPlayer src="/media/ваше-видео.mp4" title="Название видео" />
```

Все медиафайлы должны храниться в папке `public/media/` для удобного доступа.
