# MDX Showcase

Эта страница демонстрирует все основные возможности форматирования Markdown и MDX, которые можно использовать в документации AfricaRP. Текст уже увеличен, а шрифт изменен на фирменный!

## Базовое форматирование

Вы можете использовать **жирный текст**, *курсив*, ~~зачеркнутый текст~~ или `встроенный код`.

### Списки

Маркированный список:
- Первое правило AfricaRP
- Второе правило AfricaRP
  - Подпункт А
  - Подпункт Б

Нумерованный список:
1. Зайти на сервер
2. Выбрать роль
3. Начать игру

## Блоки кода (Code Hike)

Поскольку мы используем Code Hike, блоки кода выглядят просто потрясающе:

```js
// Пример простого скрипта
function greetPlayer(name) {
  console.log(`Приветствуем тебя на AfricaRP, ${name}!`);
}

greetPlayer("Steve");
```

## Спойлеры и аккордеоны

Если у нас настроены MDX-компоненты (например, Details), мы можем их использовать, но пока ограничимся стандартным HTML-тегом:

<details>
<summary>Нажмите, чтобы раскрыть секретную информацию</summary>

Здесь может быть текст, который не нужен всем подряд, но важен для некоторых игроков.

</details>
